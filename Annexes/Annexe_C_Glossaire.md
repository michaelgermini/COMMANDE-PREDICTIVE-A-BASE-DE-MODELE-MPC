# Annexe C : Glossaire

Définitions des termes clés utilisés dans ce livre.

## A

**Actionneur** : Dispositif qui applique la commande au système (moteur, vanne, etc.).

**Adaptatif** : Se dit d'un système qui s'ajuste automatiquement aux changements.

**Algorithme** : Procédure systématique pour résoudre un problème.

**Approximation** : Estimation d'une valeur ou fonction par une autre plus simple.

## B

**Bande passante** : Plage de fréquences où un système répond efficacement.

**Bruit** : Perturbation aléatoire affectant les mesures ou le système.

## C

**Commande** : Signal d'entrée appliqué au système pour le contrôler.

**Contrainte** : Limitation imposée aux variables (état, sortie, commande).

**Contrôlabilité** : Capacité d'un système à être amené à n'importe quel état.

**Critère** : Fonction évaluant la performance (à minimiser ou maximiser).

## D

**Déterminisme** : Propriété d'un système dont le comportement est prévisible.

**Discrétisation** : Conversion d'un modèle continu en modèle discret.

**Distribué** : Système réparti sur plusieurs composants communicants.

**Dynamique** : Évolution temporelle d'un système.

## E

**Échantillonnage** : Prélèvement de mesures à intervalles réguliers.

**Équilibre** : État où le système reste constant (point fixe).

**Erreur** : Différence entre valeur désirée et valeur réelle.

**Estimation** : Détermination de l'état à partir de mesures.

## F

**Faisabilité** : Propriété d'un problème d'optimisation ayant une solution.

**Feedback** : Retour d'information de la sortie vers l'entrée.

**Fonction de coût** : Fonction à minimiser dans un problème d'optimisation.

## G

**Gain** : Facteur d'amplification dans un système linéaire.

**Gradient** : Vecteur des dérivées partielles d'une fonction.

## H

**Horizon** : Période de temps considérée dans la prédiction ou l'optimisation.

**Horizon de commande** : Nombre de pas sur lesquels on optimise les commandes.

**Horizon de prédiction** : Nombre de pas sur lesquels on prédit le comportement.

**Hybride** : Système combinant dynamique continue et événements discrets.

## I

**Identification** : Détermination des paramètres d'un modèle à partir de données.

**Incrémental** : Basé sur les variations plutôt que les valeurs absolues.

**Incertitude** : Manque de connaissance exacte sur le système.

**Instabilité** : Propriété d'un système dont l'état diverge.

## J

**Jacobienne** : Matrice des dérivées partielles d'une fonction vectorielle.

## K

**Kernel** : Fonction utilisée dans les méthodes non-paramétriques (GP, SVM).

## L

**Linéaire** : Système dont la sortie est proportionnelle à l'entrée.

**Linéarisation** : Approximation d'un système non-linéaire par un système linéaire.

**Lyapunov** : Théorie permettant d'analyser la stabilité.

## M

**Matrice** : Tableau de nombres utilisé en algèbre linéaire.

**Modèle** : Représentation mathématique d'un système.

**Multi-objectif** : Problème avec plusieurs objectifs à optimiser simultanément.

**Multi-variable** : Système avec plusieurs entrées et/ou sorties.

## N

**Non-linéaire** : Système ne vérifiant pas le principe de superposition.

**Norme** : Mesure de la "taille" d'un vecteur ou matrice.

## O

**Observabilité** : Capacité de déterminer l'état à partir des sorties.

**Optimisation** : Processus de recherche du meilleur selon un critère.

**Ordre** : Nombre d'états nécessaires pour décrire un système.

## P

**Paramètre** : Valeur constante dans un modèle.

**Perturbation** : Signal externe affectant le système.

**Pondération** : Facteur d'importance dans une fonction de coût.

**Prédiction** : Estimation du comportement futur.

**Programmation** : Résolution d'un problème d'optimisation sous contraintes.

## Q

**QP (Quadratic Programming)** : Optimisation avec fonction objectif quadratique.

**Qualité** : Mesure de la performance d'un système.

## R

**Référence** : Valeur désirée pour la sortie.

**Régulateur** : Dispositif maintenant une variable à une valeur désirée.

**Robuste** : Résistant aux incertitudes et perturbations.

## S

**Stabilité** : Propriété d'un système revenant à l'équilibre après perturbation.

**Stochastique** : Aléatoire, probabiliste.

**Système** : Entité transformant des entrées en sorties.

## T

**Temps réel** : Contrainte temporelle stricte sur le calcul.

**Trajectoire** : Évolution temporelle d'un état.

## U

**Uncertainty** : Voir Incertitude.

## V

**Variable** : Quantité pouvant varier.

**Vecteur** : Tableau unidimensionnel de nombres.

## W

**Warm starting** : Initialisation d'un solveur avec la solution précédente.

## Z

**Zéro** : Valeur d'une variable de transfert annulant la sortie.

---

**Note** : Ce glossaire couvre les termes les plus importants. Pour des définitions plus détaillées, se référer aux chapitres correspondants.
